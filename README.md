# Outlast-the-monsters-game-
Strategic grid-based escape game with AI enemies, difficulty levels, leaderboards, and sound effects. Originally developed in C++, converted to modern web app. Navigate through dungeons avoiding monsters to reach the exit!

Outlast the Baddies is a strategic grid-based survival game where players must navigate their hero through a 15x40 dungeon grid to reach the exit while avoiding various enemies and obstacles. 

The game features:
1) AI-driven enemies with unique movement patterns
2) Progressive difficulty system with multiple levels
3) Real-time collision detection and pathfinding
4) Leaderboard system to track high scores
5) Dynamic sound effects using Web Audio API
6) Responsive design for desktop and mobile devices
